# 开发记录（2026-05-19）

## 项目
- 名称：Quick Tools（多功能小工具网站）
- 技术栈：React 18 + TypeScript + Vite + TailwindCSS + React Router + Zustand
- 目标部署：GitHub Pages + Cloudflare Pages

## 今日完成（功能）

### 金融类（Finance）
- 房贷计算器：输入卡片 + 结果区 + “计算”按钮 + 摊还表 + 分享链接
- 贷款计算器：输入卡片 + 结果区 + “计算”按钮 + 还款明细 + 分享链接
- 复利计算器：输入卡片 + 结果区 + “计算”按钮 + 逐年明细 + 分享链接
- 在线计算器：按键式计算器（键盘输入 + 记忆键 + %/√/1/x + 分享链接）

### 英语类（English）
- 单词接龙：新增“读音与翻译”面板；点选单词后可男声/女声朗读并翻译为中文
- 单词生成器：新增“读音与翻译”面板；生成后点选单词可朗读与翻译

## 运行方式（本机 Windows）
- Node 使用项目内 .tools/node-v20.20.2-win-x64（若系统 PATH 没有 node）
- 常用命令（在项目根目录 PowerShell）：
  - `$env:Path="c:\Users\Administrator\Desktop\小工具网站\.tools\node-v20.20.2-win-x64;" + $env:Path`
  - `npm ci`
  - `npm run dev`
  - `npm run build`
